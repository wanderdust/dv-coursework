University Manager

These visualizations are designed for the university manager or members of their support staff.

In the bubble chart the user can see the locations of UK university by number of 4-star ratings. Larger bubbles mean that there are more 4-star ratings in that area. If a user clicks on a bubble, it will show them the average 4 star ratings for each university in that area.

On the dropdown on the right, the user can select a university and compare the average marks from that university to the selected one.

If a user clicks on any of the subjects on the chart, they will see a pie chart indicating all the ratings from that university
